# S48_Sathwik_Capstone_AgriHaven

## Project Idea & Description
AgriHaven is an e-commerce platform for farmers and gardeners, offering tools, manure, seeds, irrigation systems, and pest control products. It provides a seamless shopping experience, secure payments, and a supplier marketplace while promoting sustainable farming practices and community knowledge-sharing.
